Overview

What It Does

KPP Engine is a production time-series forecasting platform built to generate multi-horizon probabilistic (quantile) forecasts at scale. At its core is a fine-tuned instance of Amazon Chronos-2, a state-of-the-art zero-shot time-series foundation model, adapted to the target domain using LoRA adapters via PEFT.

The preprocessing and feature engineering pipeline is designed to be leak-free — all transformations that depend on statistics are computed strictly on the training window during walk-forward evaluation. This eliminates lookahead bias and produces honest, reproducible accuracy numbers.

The trained model is served through a FastAPI REST API that accepts raw time-series payloads and returns calibrated quantile forecasts across configurable horizons. The entire stack — training environment, data pipeline, and serving layer — is containerized with Docker for reproducible deployments on self-hosted infrastructure.

Features

Key Components

Chronos-2 Fine-Tuning
Parameter-efficient fine-tuning of Amazon's Chronos-2 foundation model using LoRA adapters, adapting the pretrained zero-shot forecaster to the target domain with minimal compute overhead.
Probabilistic Quantile Forecasts
Outputs calibrated quantile predictions across multiple horizons rather than point estimates, providing meaningful uncertainty bounds for downstream decision-making.
Leak-Free Rolling Backtest
Walk-forward backtesting framework that recomputes all statistics within each training window, eliminating lookahead bias and producing evaluation metrics that accurately reflect live performance.
Feature Pipeline
Preprocessing pipeline that handles missing values, temporal feature engineering, scaling, and lag construction — all designed to feed cleanly into the Chronos tokenizer format.
FastAPI Inference Service
Clean REST API that accepts raw time-series payloads (JSON), runs preprocessing and model inference, and returns structured forecast responses with quantile bounds and horizon metadata.
Docker Containerization
Fully containerized training and serving environments with Docker, enabling reproducible builds and clean deployment on self-hosted infrastructure via Coolify.
